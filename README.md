### 1. Clone the Repository
```sh
git clone https://github.com/JarkkoKarki/Web-Project.git
```

### 2. Navigate to the Project Directory

```sh
cd Web-Project
```

### 3. Install Dependencies

```sh
npm install
```
### 4. Start the Development Server

```sh
npm run dev
```
